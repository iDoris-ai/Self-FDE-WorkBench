# 部署（二）· 部署你造出的专属工具

> **本文 = WorkBench 造好一个工具后，怎么把它部署、跑起来、带走。**
> 平台本身怎么架见 [DEPLOY-PLATFORM.md](./DEPLOY-PLATFORM.md)。经济/计价见 [ECONOMIC_MODEL.md](./ECONOMIC_MODEL.md)。

「造」和「部署 / 跑」是**分开**的两件事。造完，工具就是你的——怎么跑，三条路自选。

## 可携带性优先（你的数据，你做主）

任何时候都能**一键下载全部**：对话记录、追问流程、技术规格、乃至生成的代码。

- 下载后**清空** → 不留痕;下次想继续追问，**再导入**接着来。
- 或**加密留存**在机器上，下次直接加载继续。
- 拿着我们**开源**的代码，你甚至可以自己搭一套完全自持地跑。

> 这是使命层的承诺：**不锁定**。付费买便利，不买准入。

## 三条部署路径

| 路径 | 适合 | 能力 | 数据主权 | 维护 | 成本 |
|---|---|---|---|---|---|
| **① 免费 · Cloudflare** | 简单 / 较无状态的工具 | 受 CF 免费额度约束 | 你的 token/账号,数据你控 | 你自担 | **免费**（用你自己的 token） |
| **② 自持** | 想完全掌控 / 已有机器 | 取决于你的机器 | 完全你的 | 你自担 | 你的硬件 + token |
| **③ 专属 Mac Mini** | 重工具：本地模型 / 7×24 常驻 / 本地 FLUX | 最全（本地模型免 API 计费） | 完全你控,可转产权 | 我们可托管 | **$1000/月/台** |

### ① 免费 · 部署到 Cloudflare（个人默认）
我们帮你把工具部署到 Cloudflare（Pages / Workers）。**用你自己的 token 与账号**时，Cloudflare 免费额度通常够用 → **这层托管我们不收费**，属专属基础能力的一部分。
- 它背后调用的 API（Workers AI / 我们提供的 API）是**另一个服务层**，按用量另计。
- 边界：需要**本地模型、常驻任务、本地 FLUX** 的重工具，免费层接不住 → 走路径 ③。

### ② 自持
下载全部（代码 + 规格 + 数据）→ 在你自己的机器 / 云上跑我们开源的那套。想继续演进，把数据再导入 WorkBench 追问即可。

### ③ 专属 Mac Mini（重工具 / 企业）
长期租用（$1000/月）一台专属 Mac Mini：解锁本地模型（不再走计费 API）、常驻 24×7 任务、本地 FLUX，数据全程你控（含开机密码），**若干周期后产权可转给你**。对外访问经 Cloudflare Tunnel + Access（同 DEPLOY-PLATFORM.md）。

## 部署 ≠ 跑

- **部署**：把工具放上去、能访问（一次性）。
- **跑 / 托管**：让它持续运行、更新、维护（长期）。免费层你自担;托管层按月/年（含维护人力，见 ECONOMIC_MODEL §7）。

## 决策速查

- 简单工具、预算敏感 → **① 免费 Cloudflare + 自己 token**
- 想完全自持、有技术 → **② 自持（开源）**
- 要本地模型 / 常驻 / 企业级 → **③ 专属 Mac Mini（$1000/月）**
- 复杂商业场景 → ③ + **企业咨询**（见 ECONOMIC_MODEL）

> **中国大陆注意**：Cloudflare 在陆被 GFW 干扰，① 免费 Cloudflare 路径对国内访问不稳。国内改用：家庭私有访问走 **mesh VPN（Tailscale/ZeroTier）**、对外服务走 **frp + 备案 VPS（阿里云/腾讯云）**——详见 [DEPLOY-PLATFORM.md 的「中国大陆部署」](./DEPLOY-PLATFORM.md#中国大陆部署cloudflare-tunnel-在陆不可靠时)。
