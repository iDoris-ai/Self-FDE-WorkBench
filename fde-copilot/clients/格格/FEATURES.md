# Feature 细节

> 每个 feature：用户故事 + 验收标准(Given/When/Then) + 边界/异常 + 优先级
> ✅ R2 方向锁定 A（求职上岸），feature 已可拆为增量。部分参数（地域/时间线/研究方向）待客户回答后细化，见 GAPS。

## 交付路线（增量）
- **M0（诊断与地基）**：F1 竞争力画像 + F2 岗位情报底库。先让格格看清"我在哪、要去哪、差什么"。
- **M1（造弹药）**：F3 AI native 简历 + F4 作品集/教案生成。把"三无"补成"有作品"。
- **M2（打靶与临阵）**：F5 投递管理 + F6 试讲/面试演练。真正对接岗位、拿下面试。
- 每个 Feature 均可独立交付+验证。

### F1 · 竞争力画像与差距分析【M0 · P0】
**用户故事：** 作为求职中的博士，我想知道对照高职创新创业教师岗的真实要求，我现在差什么、按什么顺序补，这样我不盲目使劲。
**验收标准：**
- Given 格格已录入现状（学历/在读状态、博士研究方向、已有科研成果、语言、是否党员、年龄等），When 运行画像，Then 输出一份差距报告，含：① 满足项；② 缺口项（如"学历未认证""无双师材料""无教案"）；③ 每项的补齐动作与优先级；④ 对"泰国博士认可度"风险的应对建议。
- Given 差距报告已生成，When 格格补齐某项，Then 报告状态可更新并重算就绪度。
**边界/异常：** 岗位要求因校而异——报告需说明"基于通用高职创新创业岗画像"，具体岗以 F2 抓到的公告为准。
**依赖：** 需要 GAPS 中 Q 系列（研究方向/成果/身份）答案才能精准。

### F2 · 岗位情报聚合与匹配【M0 · P0】
**用户故事：** 作为求职者，我想只看到"博士友好、经验不限、接受海外博士"的高职创新创业岗，不想在硬门槛不符的岗上浪费时间。
**验收标准：**
- Given 已配置目标地域/意向，When 抓取招聘信息，Then 输出岗位列表，每条含：院校、岗位名、学历要求、经验要求、是否接受海外博士（若公告未写则标"待确认"）、截止日期、来源链接。
- Given 一条岗位，When 系统评估匹配度，Then 给出"匹配/存疑/不符"标签与理由（对照 F1 画像）。
**边界/异常：** 招聘信息源合规抓取；抓不到"是否接受泰博"时不臆断，标"待向院校人事确认"。
**依赖：** 目标地域（GAPS Q6）。

### F3 · AI native 简历包装【M1 · P1】
**用户故事：** 作为无企业经历的博士，我想要一份把"清迈博士 + AI native 创新创业"讲成强故事的简历，弱化产业经历短板。
**验收标准：**
- Given 格格现状 + 目标岗，When 生成简历，Then 产出针对该岗定制的简历，突出：博士学术能力、AI 融合创新创业教学的差异化、可迁移能力；不虚构不具备的经历。
- Given 简历初稿，When AI 评审，Then 给出评分与逐条改进建议，可迭代至设定分数线。
**边界/异常：** 合规底线——不得编造企业/职业经历；"无经历"用真实可迁移能力与作品替代。

### F4 · 作品集与教案生成【M1 · P1】
**用户故事：** 作为没有教案的博士，我想快速拥有几份能拿得出手的创新创业教案/课程设计/AI 教学案例，作为试讲和作品集素材。
**验收标准：**
- Given 指定课题（如"大学生创新创业基础""创业计划书实训"）与二级学院场景（会计/旅游/制造，可选），When 生成，Then 产出结构完整的教案（教学目标/重难点/教学过程/AI 工具融入点/考核），可导出可编辑文档。
- Given 一份教案，When 格格要求调整，Then 可按反馈迭代。
**边界/异常：** 输出为"初稿+可编辑"，需格格本人审校确认为自己的作品（署名与原创责任在格格）。

### F5 · 投递管理与截止提醒【M2 · P2】
**用户故事：** 作为同时投多所院校的求职者，我想统一管理每个岗位的投递状态和截止日期，别错过。
**验收标准：**
- Given 从 F2 选中的岗位，When 加入投递看板，Then 可记录状态（待投/已投/初筛/试讲/面试/offer/拒），并对临近截止的岗位提醒。
**边界/异常：** 实际投递动作由格格本人完成，系统只做记录与提醒。

### F6 · 试讲/说课与面试演练【M2 · P1】
**用户故事：** 作为要试讲的候选人，我想有 AI 评委帮我练创新创业课试讲，并练熟"为什么泰国博士""无企业经历怎么教创业"这类必被问到的问题。
**验收标准：**
- Given 一份教案/课题，When 生成试讲稿并进入模拟，Then AI 扮演评委给出提问、逐项打分（教学设计/表达/AI 融合/临场）与改进建议。
- Given 高频问题库，When 格格作答，Then AI 给出话术优化（重点：泰博认可度、无产业经历、为何选高职）。
**边界/异常：** 演练为文本/语音辅助，不替代真人试讲临场表现。
