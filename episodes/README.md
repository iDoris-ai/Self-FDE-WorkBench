## 实践方向（持续跟踪的主题）

这些是反复被提问、需要长期跟踪落地情况的核心课题：

- **Agent Memory** — 怎么解决、用什么方案、踩了什么坑
- **Model Fine-tuning** — 用什么模型、数据怎么准备、效果如何
- **Loop / Agentic Engineering** — 工作流怎么设计、如何避免死循环和无效调用
- **RAG 落地** — 检索策略、embedding 选型、实际效果差距
- **Tool Use / Function Calling** — 实际可靠性、错误处理
