## 实验目录结构

每个实验放在独立子目录下：

```
experiments/
  exp-001-tokenizer/      # 实验编号-简短描述
    README.md             # 做了什么、学到了什么
    main.py / index.ts    # 代码
```

**`README.md` 包含**：
- 想验证什么
- 怎么跑（`python main.py` 或 `node index.js`）
- 观察到了什么
- 跟预期的差异

---

*一个 50 行的脚本，如果真的验证了你的理解，比一个漂亮但没运行过的项目更有价值。*
